Controls

  • Mouse Drag - Scroll camera left or right

  • BLOCK Button - Causes Lemming to warn other Lemmings to turn around. Number above represents number of uses.

  • DIG_SDWY Button - Causes Lemming to dig sideways. Number above represents number of uses.

  • DIG_DOWN Button - Causes Lemming to dig dowmwards. Number above represents number of uses.

  • PARACHUTE Button - Causes Lemming to deploy a parachute on the next fall, allowing them to survive big falls. Number above represents number of uses.

  • SELF-DESTRUCT Button - Causes Lemming to explode after 5 seconds, destroying blocks left, down and right of said Lemming. Has infinite uses.

  • Pause Menu - Pauses the game and allows restarting the level. Note that the main menu has not been implemented yet, so the button does nothing.
